Produzioni Regole semantiche
$E → \tnum$ $E.\code = \ILDC~\tnum.v$
$E → x$ $E.\code = \IILOAD~\address{x}$
$E → E₁ \tplus E₂$ $E.\code = E₁.\code \append E₂.\code \append \IIADD$
   
$B → \ttrue$ $B.\code = \IJUMP~B.\true$
$B → \tfalse$ $B.\code = \IJUMP~B.\false$
$B → E₁~R~E₂$ $B.\code = E₁.\code \ap E₂.\code \ap \IJUMPIF{}R~B.\true \ap \IJUMP~B.\false$
$B → B₁ \tand B₂$ $B₁.\true = \newlabel()$
  $B₁.\false = B.\false$
  $B₂.\true = B.\true$
  $B₂.\false = B.\false$
  $B.\code = B₁.\code \ap B₁.\true: B₂.code$
$B → \tnot B₁$ $B₁.\true = B.\false$
  $B₁.\false = B.\true$
  $B.\code = B₁.\code$
   
$S → x = E\tsemi$ $S.\code = E.\code \ap \IISTORE~\address{x} \ap \IJUMP~S.\next$
$S → \ttIf~\tparens{B}~S₁~\ttElse~S₂$ $B.\true = \newlabel()$
  $B.\false = \newlabel()$
  $S₁.\next = S.\next$
  $S₂.\next = S.\next$
  $S.\code = B.\code \ap B.\true: S₁.\code \ap B.\false: S₂.\code$
$S → \twhile~\tparens{B}~S₁$ $B.\true = \newlabel()$
  $B.\false = S.\next$
  $S₁.\next = \newlabel()$
  $S.\code = S₁.\next : B.\code \ap B.\true : S₁.\code$
$S → \tbraces\Slist$ $\Slist.\next = S.\next$
   
$\Slist → \es$ $\Slist.\code = \IJUMP~\Slist.\next$
$\Slist → S\Slist₁$ $S.\next = \newlabel()$
  $\Slist₁.\next = \Slist.\next$
  $\Slist.\code = S.\code \ap S.\next: \Slist₁.\code$
