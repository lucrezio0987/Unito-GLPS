Grammatica dei comandi

Produzioni Descrizione
$S → x \teq E\tsemi$ Assegna il valore di $E$ ad $x$
$S → \ttIf~\tparens{B}~S₁$ Esegue $S₁$ se $B$ è vera, non fa nulla altrimenti
$S → \ttIf~\tparens{B}~S₁~\ttElse~S₂$ Esegue $S₁$ se $B$ è vera e $S₂$ altrimenti
$S → \twhile~\tparens{B}~S₁$ Esegue $S₁$ finché $B$ è vera
$S → \tdo~S₁~\twhile~\tparens{B}\tsemi$ Esegue $S₁$ finché $B$ è vera e almeno una volta
$S → \tbraces\Slist$ Esegue in sequenza i comandi in $\Slist$
   
$\Slist → \es$ Non fa nulla
$\Slist → S\Slist$ Esegue $S$ e poi i comandi in $\Slist$

Attributi

  • $S.\next$ = etichetta a cui saltare dopo che $S$ è stato eseguito (ereditato)
  • $S.\code$ = codice che esegue $S$ e salta a $S.\next$

Assegnamento

Produzioni Regole semantiche
$S → x = E\tsemi$ $S.\code = E.\code$
  ${} \ap \IISTORE~\address{x}$
  ${} \ap \IJUMP~S.\next$

Comandi condizionali

Produzioni Regole semantiche
$S → \ttIf~\tparens{B}~S₁$ $B.\true = \newlabel()$
  $B.\false = S.\next$
  $S₁.\next = S.\next$
  $S.\code = B.\code$
  ${} \ap B.\true: S₁.\code$
   
$S → \ttIf~\tparens{B}~S₁~\ttElse~S₂$ $B.\true = \newlabel()$
  $B.\false = \newlabel()$
  $S₁.\next = S.\next$
  $S₂.\next = S.\next$
  $S.\code = B.\code$
  ${} \ap B.\true: S₁.\code$
  ${} \ap B.\false: S₂.\code$

Comandi iterativi

Produzioni Regole semantiche
$S → \twhile~\tparens{B}~S₁$ $B.\true = \newlabel()$
  $B.\false = S.\next$
  $S₁.\next = \newlabel()$
  $S.\code = S₁.\next : B.\code$
  ${} \ap B.\true : S₁.\code$
   
$S → \tdo~S₁~\twhile~\tparens{B}\tsemi$ $S₁.\next = \newlabel()$
  $B.\true = \newlabel()$
  $B.\false = S.\next$
  $S.\code = B.\true: S₁.\code$
  ${} \ap S₁.\next: B.\code$

Composizione sequenziale

Produzioni Regole semantiche
$S → \tbraces\Slist$ $\Slist.\next = S.\next$
  $S.\code = \Slist.\code$
   
$\Slist → \es$ $\Slist.\code = \IJUMP~\Slist.\next$
   
$\Slist → S\Slist₁$ $S.\next = \newlabel()$
  $\Slist₁.\next = \Slist.\next$
  $\Slist.\code = S.\code$
  ${} \ap S.\next: \Slist₁.\code$

Attributi

  • $\Slist.\next$ = etichetta a cui saltare dopo che tutti i comandi in $\Slist$ sono stati eseguiti (ereditato)
  • $\Slist.\code$ = codice che esegue tutti i comandi in $\Slist$

Esempio: somma dei primi 10 numeri naturali

{
    x = 10;
    y = 0;
    while (x > 0) {
        y = x + y;
        x = x - 1;
    }
}

Note

  • L0 è l’etichetta presente nell’attributo ereditato $S.\next$ della variabile $S$ da cui è stato generato il comando.
  • Nel codice generato ci sono diversi goto ridondanti che possono essere eliminati nella fase di ottimizzazione del codice.
    ldc 10
    istore &x
    goto L1
L1: ldc 0
    istore &y
    goto L2
L2:
L3: iload &x
    ldc 0
    if_icmpgt L4
    goto L0
L4: iload &x
    iload &y
    iadd
    istore &y
    goto L5
L5: iload &x
    ldc 1
    isub
    istore &x
    goto L3
L0:
